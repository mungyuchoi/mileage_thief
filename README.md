# 마일리지도둑 (mileage_thief)

## 개요 및 목적

마일리지도둑은 항공권 마일리지 좌석을 쉽고 빠르게 검색할 수 있도록 도와주는 Flutter 기반의 앱/웹 서비스입니다. 다양한 항공사와 노선의 마일리지 좌석 정보를 한눈에 확인하고, 상세 정보를 제공하여 사용자들의 마일리지 활용을 극대화하는 것이 목적입니다.

## 주요 기능

- **항공권 마일리지 좌석 검색**  
  다양한 조건(항공사, 노선, 날짜 등)으로 마일리지 좌석을 검색할 수 있습니다.
- **상세 정보 제공**  
  검색 결과에서 각 항공권의 상세 마일리지 정보, 좌석 등급, 필요 마일리지, 세부 조건 등을 확인할 수 있습니다.

## 앱 탭 안내 (초등학생도 이해하는 쉬운 설명)

아래는 **일반 사용자 기준**으로 “무엇을 할 수 있는지”를 쉽게 풀어쓴 설명입니다.

### 하단 탭 5개

**커뮤니티 / 특가 / 상품권 / 대한항공 / 설정**

### 커뮤니티

- 사람들이 **질문하고, 정보 나누고, 이야기하는 곳**이에요.
- 왼쪽 메뉴에서 **게시판(카테고리)**를 골라서 볼 수 있어요.
- 인기 글 모음(베스트), 검색, 알림 기능이 있어요.
- 일부 게시판은 **레벨이 올라야** 들어갈 수 있어요.

### 특가

- **항공권·호텔 특가**를 모아 보는 곳이에요.
- 쇼핑몰 아이콘을 누르면 **외부 웹페이지**로 이동해요.
- 하루에 한 번, 조건을 만족하면 **땅콩 보상**을 받을 수 있어요.

### 상품권

상품권 탭은 위에 **정보 / 지도 / 시세**로 나뉘어요.

- **정보**: 내가 사고판 기록, 달력, 일일 기록, 랭킹을 볼 수 있어요.
- **지도**: 상품권 지점이 어디 있는지 지도로 보고,
  월간 랭킹도 확인할 수 있어요.
- **시세**: 오늘 **팔 때/살 때 가격**을 한눈에 보고,
  브랜드별·지점별로 자세히 확인할 수 있어요.
- **추천 기능**: 선택한 상품권에 대해
  “오늘 팔 때 가장 많이 쳐주는 지점 2곳”을 알려줘요.
  (이 기능은 땅콩이 조금 필요해요)

### 대한항공

- 대한항공 마일리지 좌석을 **편도/왕복**으로 검색할 수 있어요.
- 검색할 때 **땅콩이 소모**돼요.
- 광고를 보면 땅콩을 더 모을 수 있어요.
- **취소표 알림**도 받을 수 있어요.

### 설정

- 알림을 켜고 끄거나, 로그인/계정 정보를 관리할 수 있어요.
- 앱 버전 정보도 확인할 수 있어요.

## 커뮤니티 카테고리(게시판)와 역할

**전체글**은 모든 게시판 글을 한 번에 보는 곳이에요.

### 마일리지/혜택

- **마일리지**: 마일리지 질문, 항공사 정책, 발권 문의
- **적립/카드 혜택**: 카드 추천, 이벤트/혜택 정보  
  → 이곳은 **레벨 2 이상**부터 들어갈 수 있어요.
- **좌석 공유**: 좌석 오픈 알림, 취소표 공유
- **오늘의 좌석**: 비즈니스/퍼스트 좌석 정보  
  → 이곳은 **레벨 2 이상**부터 들어갈 수 있어요.
- **오늘의 뉴스**: 호텔·카드·마일리지 관련 소식  
  → 주로 운영진이 올려요.

### 여행/리뷰

- **항공 리뷰**: 라운지, 기내식, 좌석 후기
- **자유게시판**: 일상 이야기, 잡담, 질문

### 운영/소통

- **오류 신고**: 앱/서비스 문제 제보
- **건의사항**: 개선 요청, 아이디어 제안
- **운영 공지사항**: 운영진 공지와 업데이트 안내  
  → 주로 운영진이 올려요.

## 마이페이지와 레벨 (쉽게 설명)

마이페이지에서는 **내 정보와 활동 기록**을 볼 수 있어요.

- 내 프로필, 땅콩 개수, 레벨을 확인해요.
- 내가 쓴 글/댓글/리뷰, 좋아요, 북마크도 볼 수 있어요.

### 레벨은 어떻게 올라가요?

레벨은 **활동을 많이 하면** 올라가요.  
특히 **글쓰기, 댓글, 좋아요, 땅콩, 팔로워**가 중요해요.

#### 이코노미

- Lv.2: 글 2개 + 댓글 5개 + 좋아요 10개
- Lv.3: 글 5개 + 댓글 15개 + 좋아요 20개
- Lv.4: 글 10개 + 댓글 30개 + 좋아요 50개
- Lv.5: 글 20개 + 댓글 50개 + 좋아요 100개

#### 비즈니스

- Lv.1: 글 30개 + 댓글 70개 + 좋아요 150개
- Lv.2: 글 40개 + 댓글 100개 + 좋아요 250개
- Lv.3: 글 50개 + 댓글 150개 + 좋아요 400개
- Lv.4: 글 70개 + 댓글 200개 + 좋아요 600개
- Lv.5: 글 100개 + 댓글 300개 + 좋아요 1000개

#### 퍼스트

- Lv.1: 글 150개 + 댓글 400개 + 좋아요 1500개  
  + 땅콩 500개 + 팔로워 30명
- Lv.2: 위 조건을 모두 채우고 **운영진 승인**이 필요해요.
