# 마일리지도둑 (mileage_thief)

## 개요 및 목적

마일리지도둑은 항공권 마일리지 좌석을 쉽고 빠르게 검색할 수 있도록 도와주는 Flutter 기반의 앱/웹 서비스입니다. 다양한 항공사와 노선의 마일리지 좌석 정보를 한눈에 확인하고, 상세 정보를 제공하여 사용자들의 마일리지 활용을 극대화하는 것이 목적입니다.

## 주요 기능

- **항공권 마일리지 좌석 검색**  
  다양한 조건(항공사, 노선, 날짜 등)으로 마일리지 좌석을 검색할 수 있습니다.
- **상세 정보 제공**  
  검색 결과에서 각 항공권의 상세 마일리지 정보, 좌석 등급, 필요 마일리지, 세부 조건 등을 확인할 수 있습니다.

<h2>앱 탭 안내 (쉬운 설명)</h2>
<p>아래는 <strong>일반 사용자 기준</strong>으로 “무엇을 할 수 있는지”를 쉽게 풀어쓴 설명입니다.</p>

<h3>하단 탭 5개</h3>
<p><strong>커뮤니티 / 특가 / 상품권 / 대한항공 / 설정</strong></p>

<h3>커뮤니티</h3>
<ul>
  <li>사람들이 <strong>질문하고, 정보 나누고, 이야기하는 곳</strong>이에요.</li>
  <li>왼쪽 메뉴에서 <strong>게시판(카테고리)</strong>를 골라서 볼 수 있어요.</li>
  <li>인기 글 모음(베스트), 검색, 알림 기능이 있어요.</li>
  <li>일부 게시판은 <strong>레벨이 올라야</strong> 들어갈 수 있어요.</li>
</ul>

<h3>특가</h3>
<ul>
  <li><strong>항공권·호텔 특가</strong>를 모아 보는 곳이에요.</li>
  <li>쇼핑몰 아이콘을 누르면 <strong>외부 웹페이지</strong>로 이동해요.</li>
  <li>하루에 한 번, 조건을 만족하면 <strong>땅콩 보상</strong>을 받을 수 있어요.</li>
</ul>

<h3>상품권</h3>
<p>상품권 탭은 위에 <strong>정보 / 지도 / 시세</strong>로 나뉘어요.</p>
<ul>
  <li><strong>정보</strong>: 내가 사고판 기록, 달력, 일일 기록, 랭킹을 볼 수 있어요.</li>
  <li><strong>지도</strong>: 상품권 지점이 어디 있는지 지도로 보고, 월간 랭킹도 확인할 수 있어요.</li>
  <li><strong>시세</strong>: 오늘 <strong>팔 때/살 때 가격</strong>을 한눈에 보고, 브랜드별·지점별로 자세히 확인할 수 있어요.</li>
  <li><strong>추천 기능</strong>: 선택한 상품권에 대해 <strong>“오늘 팔 때 가장 많이 쳐주는 지점 2곳”</strong>을 알려줘요. (이 기능은 땅콩이 조금 필요해요)</li>
</ul>

<h3>대한항공</h3>
<ul>
  <li>대한항공 마일리지 좌석을 <strong>편도/왕복</strong>으로 검색할 수 있어요.</li>
  <li>검색할 때 <strong>땅콩이 소모</strong>돼요.</li>
  <li>광고를 보면 땅콩을 더 모을 수 있어요.</li>
  <li><strong>취소표 알림</strong>도 받을 수 있어요.</li>
</ul>

<h3>설정</h3>
<ul>
  <li>알림을 켜고 끄거나, 로그인/계정 정보를 관리할 수 있어요.</li>
  <li>앱 버전 정보도 확인할 수 있어요.</li>
</ul>

<h2>커뮤니티 카테고리(게시판)와 역할</h2>
<p><strong>전체글</strong>은 모든 게시판 글을 한 번에 보는 곳이에요.</p>

<h3>마일리지/혜택</h3>
<ul>
  <li><strong>마일리지</strong>: 마일리지 질문, 항공사 정책, 발권 문의</li>
  <li><strong>적립/카드 혜택</strong>: 카드 추천, 이벤트/혜택 정보<br>
    → 이곳은 <strong>레벨 2 이상</strong>부터 들어갈 수 있어요.</li>
  <li><strong>좌석 공유</strong>: 좌석 오픈 알림, 취소표 공유</li>
  <li><strong>오늘의 좌석</strong>: 비즈니스/퍼스트 좌석 정보<br>
    → 이곳은 <strong>레벨 2 이상</strong>부터 들어갈 수 있어요.</li>
  <li><strong>오늘의 뉴스</strong>: 호텔·카드·마일리지 관련 소식<br>
    → 주로 <strong>운영진이 올려요</strong>.</li>
</ul>

<h3>여행/리뷰</h3>
<ul>
  <li><strong>항공 리뷰</strong>: 라운지, 기내식, 좌석 후기</li>
  <li><strong>자유게시판</strong>: 일상 이야기, 잡담, 질문</li>
</ul>

<h3>운영/소통</h3>
<ul>
  <li><strong>오류 신고</strong>: 앱/서비스 문제 제보</li>
  <li><strong>건의사항</strong>: 개선 요청, 아이디어 제안</li>
  <li><strong>운영 공지사항</strong>: 운영진 공지와 업데이트 안내<br>
    → 주로 <strong>운영진이 올려요</strong>.</li>
</ul>

<h2>마이페이지와 레벨 (쉽게 설명)</h2>
<p>마이페이지에서는 <strong>내 정보와 활동 기록</strong>을 볼 수 있어요.</p>
<ul>
  <li>내 프로필, 땅콩 개수, 레벨을 확인해요.</li>
  <li>내가 쓴 글/댓글/리뷰, 좋아요, 북마크도 볼 수 있어요.</li>
</ul>

<h3>레벨은 어떻게 올라가요?</h3>
<p>레벨은 <strong>활동을 많이 하면</strong> 올라가요.<br>
특히 <strong>글쓰기, 댓글, 좋아요, 땅콩, 팔로워</strong>가 중요해요.</p>
