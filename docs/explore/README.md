# 마일고사 Explore 문서

작성일: 2026-05-29
업데이트: 2026-05-30

`docs/explore`는 기존 회차형 마일고사를 "마일리지 학습 RPG"로 확장하기 위한 탐색 문서 모음이다. 핵심 방향은 새 퀴즈 엔진을 따로 만드는 것이 아니라, 현재 구현된 `mockExam`의 응시/채점/랭킹/땅콩 보상 구조 위에 듀오링고형 스텝 학습 레이어를 얹는 것이다.

## 현재 제품 자산

- `mockExam/main/exams/{examId}` 기반 회차형 마일고사 데이터가 있다.
- 문제 표시 데이터와 정답표가 분리되어 있고, Cloud Functions에서 서버 채점한다.
- 응시 기록, 진행 상태, 랭킹, 공유 보상, 재도전권 구매 흐름이 이미 있다.
- 첫 응시 제출 보상은 현재 `100` 땅콩, 재도전권 구매 비용은 현재 `50` 땅콩이다.
- `BranchService.createMockExamShareLink()`로 결과 공유 링크를 만들 수 있다.

## 추천 MVP

1. `Explore` 홈을 추가해 `호텔`, `항공`, `상품권`, `카드` 같은 분야/과정을 보여준다.
2. 첫 과정은 `호텔 > Marriott`만 연다.
3. Marriott 과정 안에는 듀오링고처럼 세로 레슨 노드를 배치한다.
4. 각 레슨은 `상황 -> 개념 -> 인터랙션 문제 -> 피드백 -> 보상`을 한 화면씩 진행한다.
5. 과정 마지막의 마스터 인증은 기존 `mockExam` 회차로 연결한다.
6. UI는 웹뷰가 아니라 Flutter 네이티브 위젯으로 구현한다.
7. 학습 원천은 스사사/뉴스사사 크롤링 JSON을 RAG로 정제해 레슨 초안과 문제 아이템으로 만든다.

## 첫 메뉴 구조

초기 메뉴는 아래 4개 대주제로 시작한다.

```text
호텔
항공
카드/포인트
상품권
```

자세한 대주제/중주제/소주제 분류는 [category_taxonomy.md](category_taxonomy.md)를 기준으로 한다.

## 설계 원칙

- 사용자는 공부하러 오는 것이 아니라 실전 이득을 얻으러 온다.
- 첫 화면은 문제 목록보다 "어떤 분야/과정을 배울지"를 먼저 보여준다.
- 핵심 학습은 짧게, 실전 판단은 한 문제씩 반복해서 제공한다.
- 땅콩은 보상이자 선택권이어야 하며, 기본 학습 진입을 과하게 막지 않는다.
- 랭킹은 경쟁의 재미를 담당하고, 과정/레슨 노드는 성장의 재미를 담당한다.
