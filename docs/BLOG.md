# 마일캐치 초보자 가이드 블로그/커뮤니티 원고

이 문서는 네이버 블로그에 먼저 정리한 뒤, 마일캐치 앱의 `커뮤니티 > 사용법` 탭에 다시 올리기 위한 마스터 원고입니다. 실제 업로드 전에는 아래 이미지 자리마다 앱 화면 캡처를 붙이고, 너무 긴 경우 섹션별로 게시글을 나눠 올리면 됩니다.

## 운영 메모

- 업로드 게시판: `커뮤니티 > 사용법`
- 게시판 ID: `milecatch_guide`
- 게시판 표시명: `마일캐치 사용법`
- 대상 독자: 마일캐치를 처음 설치했거나, 기능이 많은데 어디부터 써야 할지 모르는 사용자
- 핵심 메시지: 마일캐치는 마일리지 좌석 검색, 취소표 알림, 카드/상테크 정보, 상품권 구매/판매 관리, 시세/지도/특가/랭킹을 한 앱에서 연결해주는 도구
- 추천 썸네일: 하단 탭 4개가 보이면서 상품권 탭 또는 가이드 탭 핵심 카드가 함께 보이는 화면

## 제목 후보

1. 마일캐치 처음이라면 이 글부터 보세요: 좌석 검색부터 상품권 관리까지
2. 마일캐치 사용법 총정리: 대한항공 마일리지 좌석, 취소표 알림, 상품권 기능
3. 마일캐치 초보자 가이드: 가이드/커뮤니티/상품권/프로필 탭 한 번에 이해하기
4. 상품권 상테크까지 하는 마일리지 앱, 마일캐치 제대로 시작하는 법
5. 마일캐치 기능 지도: 좌석 검색, 카드, 채팅, 상품권 시세와 특가 알림

## 이미지 삽입 규칙

- 각 섹션 첫 화면에는 기능 진입 경로가 보이는 캡처를 넣습니다.
- 입력 화면은 사용자가 따라 할 수 있도록 필드명이 보이게 캡처합니다.
- 결과 화면은 숫자, 차트, 랭킹, 지도처럼 “이 기능을 쓰면 얻는 것”이 보이게 캡처합니다.
- 개인정보, 판매 금액, 지점명, 닉네임이 민감하면 모자이크합니다.
- 커뮤니티 업로드 시 긴 원고는 3개 글로 분리할 수 있습니다.
  - 1편: 마일캐치 기본 탭, 대한항공 검색, 취소표 알림, 커뮤니티/채팅
  - 2편: 카드 기능과 내 카드 관리
  - 3편: 상품권 구매/판매, 시세, 지도, 특가, 랭킹

## 전체 캡처 체크리스트

- [ ] 홈 하단 탭 4개: `가이드`, `커뮤니티`, `상품권`, `프로필`
- [ ] 가이드 탭 퀵 액션: 카드, 채팅, 상품권, 특가, 대한항공, 구매 등록, 판매 등록, 프로필
- [ ] 대한항공 검색 화면: 편도/왕복, 출발지/도착지, 시작월/종료월, 검색하기 버튼
- [ ] 대한항공 검색 결과 화면: 날짜별 좌석, 이코노미/비즈니스/퍼스트, 세금/마일리지
- [ ] 취소표 알림 등록 화면: 구간, 좌석등급, 알림 기간, 땅콩 차감
- [ ] 취소표 알림 화면: 내 알림, 인기 알림, 알림 히스토리
- [ ] 카드 허브: 추천, 랭킹, 카드사, 이벤트
- [ ] 카드 상세: 혜택, 연회비, 적립 조건, 공유 버튼
- [ ] 내 카드 화면: 월별 카드 현황, 카드별 실적, 수동 입력
- [ ] 카드 생성 화면: cardId, 카드/카드사 이름, 카드 타입, 1마일 원가, 월 목표 실적, 메모
- [ ] 채팅 화면: 메시지 목록과 `게시글로 정리` 버튼
- [ ] 상품권 탭 상단 세그먼트: 정보, 특가, 지도, 시세, 지점
- [ ] 상품권 플로팅 버튼: 지점 생성, 카드 생성, 구매처 생성, 상품권 구매, 상품권 판매
- [ ] 상품권 구매 입력 화면: 상품권, 카드, 구매처, 결제 수단, 구매일, 액면가, 수량, 매입가, 할인율, 메모
- [ ] 상품권 판매 입력 화면: 구매 선택, 판매일, 판매가, 할인율, 수량, 지점, 예상 손익
- [ ] 상품권 정보 탭: 대시보드, 달력, 일일, 랭킹
- [ ] 상품권 특가: 브랜드 필터, 특가 카드, 특가 상세, 맞춤 알림
- [ ] 상품권 지도: 지도 마커, 지점 바텀시트, 지점 상세, 리뷰
- [ ] 상품권 시세: 브랜드별 시세, 지점별 시세, 추천 대시보드, 최근 2달 차트
- [ ] 가이드 탭 상품권 시세표: 지점/상품권 필터, 공유 카드
- [ ] 네이버 블로그 공유 후 마일캐치 게시글 작성 화면: 링크 미리보기 또는 공유 이미지 반영

# 게시글 원고

## 1. 마일캐치가 처음이라면: 이 앱은 무엇을 해주는 앱인가요?

[이미지 01: 하단 탭 4개가 보이는 홈 화면]

마일캐치는 마일리지 여행과 상테크 정보를 한곳에서 관리할 수 있게 만든 앱입니다. 처음 들어오면 하단에 네 가지 탭이 보입니다.

- `가이드`: 자주 쓰는 기능으로 빠르게 들어가는 시작 화면입니다.
- `커뮤니티`: 마일리지, 카드, 좌석, 핫딜, 사용법 글을 읽고 쓰는 공간입니다.
- `상품권`: 상품권 구매/판매 내역, 시세, 지도, 특가, 랭킹을 관리하는 핵심 탭입니다.
- `프로필`: 내 활동, 설정, 알림, 계정 정보를 보는 곳입니다.

처음 쓰는 분이라면 순서는 이렇게 잡으면 쉽습니다.

1. `가이드` 탭에서 대한항공 좌석 검색이나 카드/상품권 기능을 둘러봅니다.
2. `커뮤니티 > 사용법`에서 다른 안내 글을 확인합니다.
3. 상품권을 관리하려면 `상품권` 탭에서 카드부터 만들고 구매/판매를 기록합니다.
4. 내 활동과 알림은 `프로필`에서 확인합니다.

초보자용 한 줄 설명:

> 마일캐치는 “마일리지 좌석을 찾고, 취소표 알림을 받고, 카드와 상품권 상테크 기록까지 이어서 관리하는 앱”입니다.

## 2. 가이드 탭: 가장 먼저 눌러볼 곳

[이미지 02: 가이드 탭 퀵 액션 영역]

가이드 탭은 마일캐치의 입구입니다. 앱을 켰을 때 어디로 가야 할지 모르겠다면 여기서 시작하면 됩니다.

가이드 탭에서 바로 들어갈 수 있는 대표 기능은 다음과 같습니다.

- `카드`: 카드 혜택 DB, 카드 추천, 랭킹, 카드사별 카드 보기
- `채팅`: 실시간으로 마일캐치 사용자들이 정보를 주고받는 공간
- `상품권`: 상품권 정보/특가/지도/시세/지점 탭으로 이동
- `특가`: 항공권/호텔/쇼핑몰 적립 등 핫딜 확인
- `대한항공`: 대한항공 마일리지 좌석 검색
- `구매 등록`: 상품권 구매 내역 바로 입력
- `판매 등록`: 상품권 판매 내역 바로 입력
- `프로필`: 내 활동과 설정으로 이동

블로그 스크립트:

> 처음 설치하셨다면 가이드 탭만 먼저 보셔도 됩니다. 카드, 채팅, 상품권, 대한항공 검색처럼 자주 쓰는 기능이 한 화면에 모여 있어서, 메뉴를 찾아 헤매지 않아도 바로 이동할 수 있어요.

## 3. 대한항공 마일리지 좌석 검색

[이미지 03: 대한항공 검색 화면]

마일캐치의 대표 기능 중 하나는 대한항공 마일리지 좌석을 한 번에 검색하는 기능입니다. 대한항공 화면에서는 다음 값을 선택합니다.

- 편도 또는 왕복
- 출발지
- 도착지
- 시작월
- 종료월
- 보유 땅콩 확인

검색 비용은 앱 화면 기준으로 편도와 왕복이 다르게 표시됩니다.

- 편도 검색: 땅콩 3개
- 왕복 검색: 땅콩 5개

검색 전 확인할 것:

- 출발지와 도착지가 같은지 확인합니다.
- 도착지를 반드시 선택합니다.
- 시작월이 종료월보다 늦지 않게 설정합니다.
- 땅콩이 부족하면 광고 시청 등으로 땅콩을 모은 뒤 검색합니다.

[이미지 04: 대한항공 검색 결과 화면]

검색 결과에서는 날짜별 좌석 정보를 확인할 수 있습니다. 좌석 정보 화면에는 이코노미, 비즈니스, 퍼스트의 세금과 필요 마일리지가 표시됩니다. 원하는 날짜가 있다면 상세 정보를 보고 대한항공 앱으로 이동해 실제 발권 가능 여부를 확인하면 됩니다.

블로그 스크립트:

> 대한항공 마일리지 좌석은 날짜를 하나하나 확인하기가 번거로운데, 마일캐치에서는 출발지와 도착지, 기간만 넣고 한 번에 훑어볼 수 있습니다. 결과 화면에서 좌석 등급별 세금과 필요 마일리지를 같이 볼 수 있어 초보자도 비교하기 쉽습니다.

## 4. 취소표 알림: 원하는 구간을 등록해두기

[이미지 05: 취소표 알림 등록 화면]

마일리지 좌석은 취소표가 갑자기 나오는 경우가 많습니다. 마일캐치의 취소표 알림은 원하는 구간과 좌석등급을 저장해두고 알림을 받는 기능입니다.

등록할 때 선택하는 값:

- 출발지
- 도착지
- 좌석등급: 이코노미, 비즈니스, 퍼스트
- 알림 기간: 달력에서 날짜 또는 날짜 범위 선택
- 소모 땅콩: 좌석등급과 기간에 따라 계산

좌석등급별 기본 땅콩 계산:

- 이코노미: 1일당 1개
- 비즈니스: 1일당 2개
- 퍼스트: 1일당 5개

여러 좌석등급을 동시에 선택하면 합산됩니다. 예를 들어 비즈니스와 퍼스트를 3일 동안 등록하면 하루 7개씩, 총 21개의 땅콩이 필요합니다.

[이미지 06: 취소표 알림 내 알림 화면]

알림을 등록한 뒤에는 `취소표 알림 > 내 알림`에서 조건을 확인할 수 있습니다. 알림 기간, 좌석등급, 구간, 만료 여부, 알림 히스토리를 확인하고 필요 없는 알림은 삭제할 수 있습니다.

[이미지 07: 취소표 인기 알림 화면]

인기 알림 탭에서는 많은 사람들이 관심을 가진 구간도 확인할 수 있습니다. 처음 어디를 검색할지 모르겠다면 인기 구간을 참고해도 좋습니다.

블로그 스크립트:

> 원하는 좌석이 지금 없다고 끝이 아닙니다. 취소표 알림에 구간과 좌석등급을 저장해두면, 나중에 조건에 맞는 좌석이 잡혔을 때 확인하기 훨씬 편해집니다.

## 5. 카드 기능: 마일리지와 상테크의 기준값 만들기

[이미지 08: 카드 허브 화면]

가이드 탭의 `카드`를 누르면 카드 허브로 이동합니다. 카드 허브에는 네 가지 탭이 있습니다.

- 추천: 내 조건에 맞는 카드 추천
- 랭킹: 인기 카드 확인
- 카드사: 카드사별 카드 목록
- 이벤트: 카드 관련 이벤트 확인

카드 추천에서는 월 사용액, 상테크 금액, 1마일 가치, 선호 항공사 같은 조건을 기준으로 예상 월 마일, 연 마일 가치, 연 순가치 등을 비교할 수 있습니다.

[이미지 09: 카드 상세 화면]

카드 상세에서는 카드명, 카드사, 연회비, 마일 적립 조건, 혜택 내용을 확인합니다. 마음에 드는 카드는 공유할 수 있고, 커뮤니티에서 카드 정보를 참고할 수도 있습니다.

[이미지 10: 내 카드 화면]

상품권 기능을 제대로 쓰려면 `내 카드` 또는 상품권 탭의 `카드 생성`에서 내 카드 규칙을 먼저 만들어두는 것이 좋습니다.

내 카드 화면에서 볼 수 있는 것:

- 월별 카드 현황
- 카드 개수
- 실적 인정 금액
- 적립 마일
- 제외 금액
- 카드별 현황
- 수동 입력

[이미지 11: 카드 생성 화면]

카드 생성 시 입력하는 값:

- `cardId`: 소문자, 숫자, `_`로 만든 카드 식별자
- 카드/카드사 이름
- 카드 타입: 신용카드 또는 체크카드
- 신용카드 1마일 원가
- 체크카드 1마일 원가
- 월 목표 실적
- 메모

상품권을 살 때 어떤 카드로 결제했는지 연결하려면 카드 규칙이 필요합니다. 그래서 상품권 기능을 시작하는 첫 단계는 카드 생성입니다.

블로그 스크립트:

> 상품권 구매/판매 기록은 결국 “얼마에 샀고, 어떤 카드로 결제했고, 마일 원가는 얼마인가”를 알아야 의미가 있습니다. 그래서 마일캐치에서는 카드 규칙을 먼저 만들어두면 이후 구매/판매 기록과 대시보드가 훨씬 정확해집니다.

## 6. 채팅 기능: 대화에서 바로 게시글로 정리하기

[이미지 12: 채팅 화면]

가이드 탭이나 상단 채팅 버튼을 누르면 마일캐치 채팅으로 들어갈 수 있습니다. 채팅은 실시간 정보, 질문, 짧은 제보를 빠르게 나누는 곳입니다.

채팅에서 유용한 기능:

- 텍스트 메시지 작성
- 이미지 공유
- 여러 메시지 선택
- 선택한 메시지를 `게시글로 정리`
- 신고 기능

[이미지 13: 채팅 메시지 선택 후 게시글로 정리 버튼]

채팅에서 좋은 정보가 지나가면 그냥 흘러가기 쉽습니다. 이때 메시지를 선택하고 `게시글로 정리`를 누르면, 선택한 메시지가 커뮤니티 글 초안으로 만들어집니다. 게시판을 선택한 뒤 제목과 내용을 다듬어 등록하면 됩니다.

블로그 스크립트:

> 채팅은 빠르고, 커뮤니티 글은 오래 남습니다. 좋은 정보가 채팅에 올라왔을 때는 메시지를 선택해서 게시글로 정리해두면 나중에 다른 사용자도 쉽게 찾아볼 수 있습니다.

## 7. 커뮤니티 사용법 탭에 글 올리기

[이미지 14: 커뮤니티 사용법 탭]

커뮤니티에는 여러 게시판이 있습니다. 그중 사용법 글은 `사용법` 탭에 올리면 됩니다.

대표 게시판:

- 전체글
- 자유게시판
- 적립/카드
- 사용법
- 핫딜
- 마일리지
- 오늘의 좌석
- 오늘의 뉴스
- 운영 공지사항

마일캐치 사용법 글을 올릴 때 추천 구조:

1. 제목에 대상과 기능을 명확히 적습니다.
2. 첫 문단에서 “누구를 위한 글인지”를 말합니다.
3. 기능마다 진입 경로를 먼저 씁니다.
4. 캡처 이미지를 넣습니다.
5. 입력값과 결과 화면을 순서대로 설명합니다.
6. 마지막에 관련 기능 링크나 다음 글 예고를 넣습니다.

예시 제목:

> 마일캐치 처음 쓰는 분들을 위한 상품권 탭 사용법

## 8. 상품권 탭 전체 구조

[이미지 15: 상품권 탭 상단 세그먼트]

상품권 탭은 마일캐치에서 가장 디테일하게 만들어진 기능입니다. 상단에는 다섯 개 탭이 있습니다.

- `정보`: 내 상품권 구매/판매 내역, 대시보드, 달력, 일일 내역, 랭킹
- `특가`: 온라인 상품권 특가, 구매 링크, 맞춤 알림
- `지도`: 지점 위치, 월별 판매왕, 지점 랭킹, 지점 상세
- `시세`: 브랜드별 시세, 지점별 시세, 추천 대시보드, 시세 추이
- `지점`: 지점 목록과 상세 정보

[이미지 16: 상품권 탭 플로팅 버튼]

상품권 탭의 `정보` 화면에서 플로팅 버튼을 누르면 다음 기능으로 바로 이동할 수 있습니다.

- 지점 생성
- 카드 생성
- 구매처 생성
- 상품권 구매
- 상품권 판매

초보자 추천 흐름:

1. 카드 생성
2. 구매처 생성
3. 상품권 구매 입력
4. 상품권 판매 입력
5. 정보 탭에서 대시보드 확인
6. 시세/지도/특가로 다음 구매와 판매 판단

블로그 스크립트:

> 상품권 탭은 처음 보면 기능이 많아 보이지만 순서는 단순합니다. 먼저 카드를 만들고, 어디서 샀는지 구매처를 만들고, 구매 내역을 넣고, 판매할 때 판매 내역을 입력하면 됩니다. 그러면 대시보드와 랭킹, 달력, 일일 기록이 자동으로 의미 있는 데이터가 됩니다.

## 9. 상품권 1단계: 카드 생성

[이미지 17: 상품권 탭에서 카드 생성 진입]

상품권을 사기 전에 먼저 카드 규칙을 만듭니다. 카드 규칙은 나중에 마일 원가와 카드별 수익률을 계산하는 기준이 됩니다.

입력 예시:

- cardId: `shinhan_purple`
- 카드/카드사 이름: `신한카드 퍼플`
- 카드 타입: 신용카드
- 신용카드 1마일 원가: `15`
- 체크카드 1마일 원가: 비워두거나 해당 값 입력
- 월 목표 실적: `1000000`
- 메모: 상품권 실적 인정 여부, 주의사항

초보자에게 설명할 포인트:

- cardId는 내부 구분값이라 짧고 알아보기 쉽게 적습니다.
- 1마일 원가는 카드별 효율 비교에 사용됩니다.
- 월 목표 실적을 넣어두면 내 카드 현황에서 관리하기 편합니다.

## 10. 상품권 2단계: 구매처 생성

[이미지 18: 구매처 생성 화면]

구매처는 내가 상품권을 어디서 샀는지 기록하는 용도입니다. 온라인몰, 오프라인 매장, 이벤트 페이지 등 자주 쓰는 구매처가 있다면 만들어두면 좋습니다.

구매처를 만들어두면 좋은 이유:

- 구매 내역 입력 시 선택이 빨라집니다.
- 나중에 어디서 샀는지 추적하기 쉽습니다.
- 같은 조건의 구매를 반복할 때 실수를 줄일 수 있습니다.

블로그 스크립트:

> 상품권을 여러 곳에서 사다 보면 “이건 어디서 산 거였지?”가 금방 헷갈립니다. 구매처를 만들어두면 나중에 내역을 다시 볼 때 훨씬 편합니다.

## 11. 상품권 3단계: 구매 정보 입력

[이미지 19: 상품권 구매 입력 화면]

상품권 구매 화면은 내가 상품권을 산 기록을 남기는 곳입니다. 이 단계가 정확해야 나중에 수익, 마일, 원가 계산이 맞아집니다.

입력하는 값:

- 상품권: 롯데, 신세계, 현대, 갤러리아 등
- 카드: 미리 만들어둔 카드 규칙
- 구매처: 선택 사항
- 결제 수단: 신용 또는 체크
- 구매일
- 액면가
- 수량
- 매입가: 권당 구매 가격
- 할인율
- 합계
- 메모

[이미지 20: 상품권 구매 템플릿 화면]

자주 쓰는 구매 조건은 템플릿으로 저장할 수 있습니다. 같은 상품권, 같은 카드, 비슷한 가격으로 반복 구매한다면 템플릿을 만들어두는 것이 좋습니다.

초보자 입력 예시:

- 상품권: 신세계상품권
- 카드: 신한카드
- 구매처: 네이버스토어
- 결제 수단: 신용
- 구매일: 오늘 날짜
- 액면가: 100,000원
- 수량: 10장
- 매입가: 97,000원
- 할인율: 3.0%
- 메모: 이벤트 적립 확인 필요

블로그 스크립트:

> 구매 입력은 “내가 무엇을, 어떤 카드로, 얼마에, 몇 장 샀는지”를 기록하는 단계입니다. 여기까지 입력해두면 나중에 판매할 때 어떤 구매분을 팔았는지 연결할 수 있습니다.

## 12. 상품권 4단계: 판매 정보 입력

[이미지 21: 상품권 판매 입력 화면]

구매한 상품권을 판매했다면 `상품권 판매` 화면에서 판매 내역을 입력합니다.

입력하는 값:

- 구매 선택(Lot): 기존에 입력한 구매 내역 중 어떤 물량을 팔지 선택
- 판매일
- 판매가: 권당 판매 가격
- 할인율
- 판매 수량
- 지점: 선택 사항
- 총 판매금액
- 총 매입금액
- 예상 손익

판매 화면에서 중요한 점:

- 판매할 수량은 선택한 구매 물량 안에서 입력합니다.
- 지점을 선택하면 지점별/월별 랭킹과 지도 기능에 연결됩니다.
- 판매가와 매입가 차이를 통해 예상 손익을 바로 볼 수 있습니다.

[이미지 22: 판매 저장 후 정보 탭 반영 화면]

판매 저장 후에는 정보 탭의 대시보드, 달력, 일일 내역, 랭킹에 반영됩니다.

블로그 스크립트:

> 상품권 판매 입력은 단순히 판매 금액만 적는 게 아니라, “어떤 구매분을 어느 지점에서 얼마에 팔았는지”를 연결하는 단계입니다. 이 연결이 되어야 대시보드와 랭킹, 지도 기능이 살아납니다.

## 13. 상품권 정보 탭: 대시보드, 달력, 일일, 랭킹

[이미지 23: 상품권 정보 탭 대시보드]

상품권 정보 탭은 내 기록을 분석해주는 화면입니다. 상단에는 네 가지 세부 탭이 있습니다.

- 대시보드
- 달력
- 일일
- 랭킹

대시보드에서 볼 수 있는 대표 지표:

- 총 구매금액
- 총 판매금액
- 총 수익
- 누적 마일
- 평균마일원가(원/마일)
- 미판매 수량
- 브랜드별 분포
- 브랜드별 잔여 소진율
- 일별 매입/판매/마일 추세
- 브랜드별 매입 대비 매출 ROI
- 카드별 평균 수익률 비교
- 월별 손익/판매/마일 추세

[이미지 24: 상품권 달력 화면]

달력에서는 구매와 판매가 날짜별로 어떻게 찍혔는지 볼 수 있습니다. “이번 달에 언제 많이 샀는지, 언제 많이 팔았는지”를 빠르게 파악할 수 있습니다.

[이미지 25: 상품권 일일 화면]

일일 탭에서는 날짜별 구매/판매 기록을 더 자세히 확인합니다. 상품권 필터를 사용하면 특정 상품권만 골라서 볼 수 있습니다.

[이미지 26: 상품권 랭킹 화면]

랭킹 탭에서는 사용자별 판매 랭킹을 확인할 수 있습니다. 판매 지점을 입력해둔 내역이 많을수록 랭킹과 지도 데이터가 풍성해집니다.

블로그 스크립트:

> 상품권 정보 탭은 내가 입력한 기록을 그냥 목록으로 보여주는 곳이 아닙니다. 구매, 판매, 마일, 손익, 카드 효율, 브랜드별 흐름을 한 번에 보는 대시보드입니다. 기록을 꾸준히 넣을수록 내가 어떤 상품권과 카드 조합에서 효율이 좋은지 보이기 시작합니다.

## 14. 상품권 특가: 조건에 맞는 상품권을 빠르게 찾기

[이미지 27: 상품권 특가 목록 화면]

상품권 특가 탭에서는 온라인에서 확인된 상품권 특가를 모아볼 수 있습니다. 상단 브랜드 필터를 이용해 전체, 현대, 롯데, 신세계 등 원하는 브랜드만 볼 수 있습니다.

특가 카드에서 확인할 수 있는 것:

- 상품권 이름
- 판매처
- 구매가
- 할인율
- 구매 버튼
- 공유 버튼

[이미지 28: 상품권 특가 상세 화면]

특가 상세에서는 가격 정보와 가격 이력 차트를 확인할 수 있습니다. 구매 페이지 열기를 누르면 외부 구매 링크로 이동합니다.

[이미지 29: 상품권 맞춤 알림 설정 화면]

맞춤 알림에서는 조건에 맞는 특가가 처음 나오거나 더 좋아질 때 알림을 받을 수 있습니다.

설정할 수 있는 조건:

- 알림 이름
- 상품권 브랜드
- 권종
- 상점
- 특정 딜
- 최소 할인율
- 최대 구매가

빠른 추천 예시:

- 현대 2% 이상
- 신세계 10만원권
- 롯데/현대 50만원권
- 현재 딜만

블로그 스크립트:

> 상품권 특가의 핵심은 “내가 원하는 조건만 알림 받는 것”입니다. 브랜드, 권종, 상점, 할인율을 설정해두면 조건을 만족하는 딜이 처음 나오거나 더 좋아질 때만 알려줍니다.

## 15. 상품권 지도: 지점과 판매왕을 한눈에 보기

[이미지 30: 상품권 지도 화면]

지도 탭에서는 상품권 지점을 지도 위에서 볼 수 있습니다. 위치 권한을 허용하면 내 주변 지점 확인이 더 편합니다.

지도에서 볼 수 있는 것:

- 지점 마커
- 지점명
- 연락처
- 주소
- 월별 판매왕
- 지점별 랭킹
- 사용자 랭킹
- 공식 인증 지점 표시

[이미지 31: 지도 지점 바텀시트]

마커를 누르면 지점 정보와 월별 랭킹이 바텀시트로 열립니다. 공식 인증 지점은 인증 표시가 보이며, 상세 화면으로 이동해 리뷰를 쓸 수도 있습니다.

[이미지 32: 지점 상세/리뷰 화면]

지점 상세에서는 주소, 전화번호, 영업시간, 안내사항, URL, 리뷰를 확인할 수 있습니다.

블로그 스크립트:

> 상품권 지도는 단순한 지도 기능이 아닙니다. 어느 지점에서 누가 많이 팔았는지, 어떤 지점이 활발한지, 인증 지점인지까지 함께 볼 수 있습니다. 판매 내역을 입력할 때 지점을 선택해두면 지도와 랭킹이 더 의미 있게 쌓입니다.

## 16. 상품권 시세: 팔 때와 살 때를 비교하기

[이미지 33: 상품권 시세 브랜드 목록]

시세 탭에서는 상품권 브랜드별 현재 시세를 확인할 수 있습니다. 각 브랜드 카드에는 `팔 때`와 `살 때` 기준 가격이 표시됩니다.

시세 화면에서 보는 값:

- 팔 때 최대 가격
- 팔 때 최소 가격
- 살 때 최소 가격
- 살 때 최대 가격
- 할인율 또는 시세율

용어 설명:

- 팔 때: 내가 지점에 상품권을 팔고 돈을 받는 경우
- 살 때: 내가 지점에서 상품권을 사는 경우

[이미지 34: 브랜드별 지점 시세 화면]

브랜드를 누르면 해당 상품권을 취급하는 지점별 현재 시세를 볼 수 있습니다. 어느 지점이 가장 유리한지 비교할 때 사용합니다.

[이미지 35: 추천 대시보드 상품권 선택 화면]

추천 대시보드는 선택한 상품권 기준으로 오늘 팔 때 가장 유리한 지점 Top 2를 비교해줍니다. 선택한 상품권 개수에 따라 땅콩이 차감됩니다.

- 선택 상품권 1개: 땅콩 3개
- 선택 상품권 2개: 땅콩 6개
- 선택 상품권 3개: 땅콩 9개

[이미지 36: 지점 상세 시세와 최근 2달 차트]

지점 상세 시세 화면에서는 지점 정보와 함께 이 지점에서 취급하는 상품권 목록, 팔 때/살 때 가격, 최근 2달 시세 추이를 볼 수 있습니다.

블로그 스크립트:

> 시세 탭은 “어디에 팔면 가장 이득일까?”를 판단하는 화면입니다. 브랜드별로 먼저 보고, 특정 상품권을 눌러 지점별 시세를 비교하고, 필요하면 추천 대시보드로 Top 지점을 빠르게 확인하면 됩니다.

## 17. 가이드 탭의 상품권 시세표 공유

[이미지 37: 가이드 탭 상품권 시세표]

가이드 탭에는 상품권 시세표를 한눈에 볼 수 있는 영역이 있습니다. 이 표는 내가 관심 있는 지점과 상품권만 골라서 볼 수 있습니다.

설정 가능한 값:

- 지점: 최대 3개
- 상품권: 최소 3개, 최대 6개
- 표시 순서

설정이 없을 때는 기본 지점과 기본 상품권이 표시됩니다. 내가 자주 보는 지점과 상품권으로 바꾸면 매번 더 빠르게 확인할 수 있습니다.

[이미지 38: 상품권 시세표 설정 화면]

상품권 시세표 설정에서는 지점과 상품권을 선택합니다. 예를 들어 롯데, 신세계, 현대 상품권을 자주 본다면 이 세 개를 우선으로 배치하면 됩니다.

[이미지 39: 시세표 공유 카드]

공유 버튼을 누르면 이미지와 텍스트로 공유할 수 있습니다. 커뮤니티, 블로그, 채팅방에 “오늘 시세표”처럼 올리기 좋습니다.

블로그 스크립트:

> 가이드 탭의 시세표는 매일 확인하는 사람에게 특히 편합니다. 내가 보는 지점 1~3개와 상품권 3~6개를 정해두면, 앱을 열자마자 필요한 시세만 바로 볼 수 있고 공유 카드로도 보낼 수 있습니다.

## 18. 네이버 블로그에 먼저 쓰고 마일캐치 게시글로 올리는 방법

[이미지 40: 네이버 블로그 글 작성 화면]

긴 가이드 글은 네이버 블로그에서 먼저 정리해도 좋습니다. 캡처 이미지를 넣고, 문단을 충분히 다듬은 뒤 마일캐치 커뮤니티로 가져오면 됩니다.

추천 작업 순서:

1. 네이버 블로그에 원고를 작성합니다.
2. 앱 화면 캡처를 각 섹션에 붙입니다.
3. 제목과 첫 문단을 검색 친화적으로 정리합니다.
4. 블로그 글을 발행합니다.
5. 블로그 글 또는 공유할 내용을 휴대폰 공유 메뉴로 보냅니다.
6. 공유 대상에서 마일캐치를 선택합니다.
7. 마일캐치 게시글 작성 화면이 열리면 게시판을 `사용법`으로 바꿉니다.
8. 링크 미리보기, 이미지, 본문을 확인합니다.
9. 제목과 본문을 커뮤니티 톤에 맞게 한 번 더 줄입니다.
10. 등록합니다.

[이미지 41: 공유 후 마일캐치 게시글 작성 화면]

마일캐치 작성 화면에서는 공유된 텍스트, 제목, 이미지가 초안에 들어갈 수 있습니다. 본문 안에 URL이 있으면 링크 미리보기 카드가 생성됩니다. 네이버 블로그 링크를 붙이면 제목, 설명, 대표 이미지가 카드 형태로 보일 수 있습니다.

커뮤니티용으로 다듬을 때 체크할 것:

- 게시판이 `사용법`인지 확인
- 제목이 너무 길면 40자 안팎으로 줄이기
- 블로그 말투보다 앱 사용자에게 바로 말하는 톤으로 바꾸기
- 이미지가 너무 많으면 핵심 화면만 남기기
- 개인정보와 판매금액 노출 확인
- 링크 미리보기 카드가 잘 보이는지 확인

블로그 스크립트:

> 저는 긴 글은 네이버 블로그에서 먼저 정리하고, 이후 마일캐치 커뮤니티 사용법 탭에 다시 올리는 방식을 추천합니다. 블로그에서는 검색 유입을 노리고, 커뮤니티에서는 앱 사용자들이 바로 따라 할 수 있게 핵심 화면과 순서를 더 짧게 정리하면 좋습니다.

## 19. 초보자에게 추천하는 첫날 사용 순서

[이미지 42: 초보자 추천 루트 요약 이미지]

처음 들어온 사용자는 이 순서대로만 해도 마일캐치의 핵심을 대부분 이해할 수 있습니다.

1. `가이드` 탭에서 전체 기능을 훑어봅니다.
2. `대한항공`에서 원하는 구간을 한 번 검색해봅니다.
3. 좌석이 없으면 `취소표 알림`을 등록합니다.
4. `카드`에서 카드 허브와 추천 기능을 봅니다.
5. 상품권을 쓸 예정이면 `상품권 > 정보`에서 카드 생성을 먼저 합니다.
6. 구매처를 만들고 `상품권 구매`를 입력합니다.
7. 판매 후 `상품권 판매`를 입력합니다.
8. 대시보드, 달력, 일일 내역, 랭킹을 확인합니다.
9. 다음 구매 전에는 `특가`, `지도`, `시세`를 확인합니다.
10. 궁금한 점은 채팅이나 커뮤니티에 남깁니다.

초보자용 마무리 문구:

> 마일캐치는 처음엔 기능이 많아 보이지만, 실제 흐름은 “검색하고, 알림 받고, 기록하고, 비교하는 것”입니다. 좌석 검색으로 여행 기회를 찾고, 상품권 기록으로 내 상테크 흐름을 관리해보세요.

## 20. 커뮤니티 업로드용 짧은 버전

아래는 커뮤니티에 바로 올릴 때 사용할 수 있는 축약 버전입니다.

```text
마일캐치 처음 쓰는 분들을 위한 빠른 가이드입니다.

1. 가이드 탭
- 카드, 채팅, 상품권, 특가, 대한항공, 구매 등록, 판매 등록으로 바로 이동할 수 있습니다.

2. 대한항공 좌석 검색
- 편도/왕복, 출발지/도착지, 시작월/종료월을 고르고 검색합니다.
- 결과에서 이코노미/비즈니스/퍼스트 좌석과 세금/마일리지를 확인합니다.

3. 취소표 알림
- 원하는 구간, 좌석등급, 날짜를 등록해두면 취소표를 놓치지 않기 좋습니다.

4. 카드
- 상품권 기능을 쓰기 전 카드 규칙을 먼저 만들어두면 원/마일, 실적, 카드별 효율 계산이 쉬워집니다.

5. 상품권
- 추천 순서: 카드 생성 → 구매처 생성 → 상품권 구매 입력 → 상품권 판매 입력 → 대시보드 확인
- 정보 탭에서는 대시보드/달력/일일/랭킹을 볼 수 있습니다.
- 특가 탭에서는 브랜드별 특가와 맞춤 알림을 설정할 수 있습니다.
- 지도 탭에서는 지점, 판매왕, 랭킹, 리뷰를 볼 수 있습니다.
- 시세 탭에서는 팔 때/살 때 가격과 지점별 시세, 최근 2달 차트를 확인할 수 있습니다.

6. 블로그 글 가져오기
- 네이버 블로그에 먼저 정리한 뒤 공유 메뉴로 마일캐치에 보내면 게시글 초안으로 활용할 수 있습니다.
- 사용법 탭에 올릴 때는 제목, 이미지, 링크 미리보기를 확인하고 등록하면 됩니다.
```

## 21. 후속 글로 나누는 경우 추천 목차

1편 제목:

> 마일캐치 처음 시작하기: 가이드 탭, 대한항공 좌석 검색, 취소표 알림

1편 구성:

- 하단 탭 설명
- 가이드 탭 퀵 액션
- 대한항공 검색
- 취소표 알림
- 채팅과 커뮤니티

2편 제목:

> 마일캐치 카드 기능 사용법: 카드 추천부터 내 카드 관리까지

2편 구성:

- 카드 허브
- 카드 추천
- 카드 랭킹/카드사/이벤트
- 카드 상세
- 카드 생성
- 내 카드 대시보드

3편 제목:

> 마일캐치 상품권 탭 완전정복: 구매/판매, 특가, 지도, 시세, 랭킹

3편 구성:

- 상품권 탭 구조
- 카드 생성/구매처 생성
- 구매 입력
- 판매 입력
- 정보 탭 대시보드
- 특가와 맞춤 알림
- 지도와 지점 랭킹
- 시세와 추천 대시보드
- 가이드 탭 시세표 공유
